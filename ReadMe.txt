Краткая информация: 
 Запуская приложение, Вы попадете на главную страницу, на которой будет выведен список всех торрентов на сайте. 
 В шапке сайте можно выбрать интересующею категорию, после чего, будет показан список торрентов данной категории. 
 На сайте присутствуют возможности: регистрации, скачивания торрентов(для зарегестрированых пользователей), добавления торрентов(для Admin'ов). 
 Так же есть возможность оставлять комментарии к тореренту, и редактировать их. 
 Зарегистрированные пользователи могут отправлять приватные сообщения других юзерам, стоит всего лишь зайди на профиль интересующего юзера, и отправить сообщение. 
 Есть фильтрация по входящим и исходящим сообщениям. 
 Пользователь так же может редактировать некоторые данные своего профиля. 
 На странице конкретного торрента, есть блок, в котором показаны Топ 6 популярных торрентов по скачиванию.
